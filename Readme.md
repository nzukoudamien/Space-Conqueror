A 2D text-based space shooter game written in **pure Lua** using Object-Oriented Programming (OOP) concepts.

## 2. Why This Project?

To finalize my foundational training in Lua, I wanted to build something fun and straightforward. The project successfully runs (though it is not perfect, I plan to improve it day by day). 

Later on, I realized that this game shares the exact same core logic as **Blastar**, the very first space game coded by Elon Musk in 1984 when he was just 12 years old!

## 3. How to Play & Commands

### 📦 Prerequisites
You only need **Lua** installed on your machine (Windows, Linux, or macOS).

### 🚀 Launching the Game
Open your terminal in the project directory and run:
```bash
lua main.lua
⌨️ Controls
q + Enter : Move Left

d + Enter : Move Right

f + Enter : Shoot Laser

Enter (without typing anything) : Skip turn / Stay still
